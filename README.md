# New Age Skills

A lightweight interactive roadmap for no-degree career tracks, built with HTML, CSS, and JavaScript.

## Contents

- `index.html` — main landing page
- `styles.css` — visual styling and layout
- `script.js` — dynamic module rendering and dropdown behavior
- `.gitignore` — local ignore rules

## Features

- Expandable skill modules
- Collapsible resource tracks
- Clickable links for free and paid learning resources
- Light, readable UI for fast content access

## Setup

Open `index.html` in a browser, or serve the folder from any static host.

## Git

The project is already initialized as a Git repository locally.

## Notes

- `node` is not required to view the page; it runs entirely in the browser.
- The repo remote should be configured to `https://github.com/DNVcoder26/index.html.git` once the GitHub repo exists.
